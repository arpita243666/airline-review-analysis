# ✈️ Airline Review Analysis

## 📌 Overview
This project analyzes airline customer reviews to uncover insights about passenger satisfaction using Python.

## 📊 Key Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Data Visualization
- Statistical Analysis (Shapiro Test, T-test, Chi-square)

## 📈 Visualizations
- Top Airlines by Reviews
- Rating Distribution (Histogram + Normal Curve)
- Boxplot (Outlier Detection)
- Correlation Heatmap
- Geo-style Scatter Plot

## 🛠 Tech Stack
Python | Pandas | NumPy | Matplotlib | Seaborn | SciPy

## 📊 Key Insights
- Ratings vary significantly across airlines
- Customer satisfaction depends on multiple factors
- Data shows presence of outliers
- Weak correlations indicate complex behavior
